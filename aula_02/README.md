# 📚 Exercícios de Banco de Dados MySQL  

Repositório contendo atividades práticas desenvolvidas para aprendizado de **Banco de Dados MySQL** Genetarion Brasil, abordando conceitos de criação de bancos, tabelas, inserção de dados, consultas e atualizações.

## 🛠️ Tecnologias Utilizadas

* 🐬 MySQL Server
* 🖥️ MySQL Workbench
* 💾 SQL

---

# 👨‍💼 Atividade 1: Serviço de RH

### 📌 Contexto

Criação de um banco de dados (**db_rh**) para gerenciamento de colaboradores.

### ✅ O que foi desenvolvido

* Criação da tabela **tb_colaboradores**
* Definição dos atributos:

  * ID
  * Nome
  * CPF
  * Cargo
  * Salário
* Inserção de **5 colaboradores**
* Consultas utilizando `SELECT`:

  * Salários maiores que **R$ 2.000,00**
  * Salários menores que **R$ 2.000,00**
* Atualização de dados utilizando `UPDATE`

---

# 🛒 Atividade 2: E-commerce

### 📌 Contexto

Criação do banco de dados (**db_ecommerce**) para controle de produtos e estoque.

### ✅ O que foi desenvolvido

* Criação da tabela **tb_produtos**
* Definição dos atributos:

  * Nome
  * Preço
  * Categoria
  * Quantidade em Estoque
* Inserção de **8 produtos**
* Consultas utilizando `SELECT`:

  * Produtos com valor acima de **R$ 500,00**
  * Produtos com valor abaixo de **R$ 500,00**
* Atualização de informações de produtos com `UPDATE`

---

# 🎓 Atividade 3: Registro Escolar

### 📌 Contexto

Criação do banco de dados (**db_escola**) para gerenciamento de estudantes.

### ✅ O que foi desenvolvido

# 🚀 Como Executar os Scripts

### 1️⃣ Clone este repositório

```bash
git clone https://github.com/Dev-Everly/bd_mysql.git
```

### 2️⃣ Abra o MySQL Workbench

Conecte-se à sua instância local do MySQL.

### 3️⃣ Abra um script SQL

Acesse:

```text
File > Open SQL Script...
```

Selecione o arquivo desejado:

```text
exercicio01_rh.sql
exercicio02_ecommerce.sql
exercicio03_escola.sql
```

### 4️⃣ Execute o script

Clique no ícone ⚡ ou utilize o atalho:

```text
Ctrl + Shift + Enter
```
